# Brainspill

Brainspill is a witty, low-chrome infinite canvas app for dumping thoughts, images, notes, and visual ideas into a flexible creative workspace.

## Current Status

The app is in Milestone 1:

- Next.js App Router scaffold
- shadcn/ui setup
- Local board dashboard
- Dynamic board workspace route
- Local board persistence
- Typed local canvas scene model

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Verification

```bash
npm run lint
npm run build
```
