# Brainspill Docs

This folder contains the planning documents for Brainspill, an infinite canvas webapp.

## Documents

- [Context](context.md): product background, users, assumptions, and non-goals.
- [Scope](scope.md): MVP, V1, V2, deferred features, and success criteria.
- [Design](design.md): UX principles, canvas interactions, visual thesis, and object behavior.
- [Architecture](architecture.md): Next.js, Convex, Yjs, auth, storage, and rendering architecture.
- [Data Model](data-model.md): product records, Yjs document shape, assets, and permissions.
- [Realtime Collaboration](realtime-collaboration.md): Yjs collaboration model, presence, conflict behavior, and persistence.
- [Epics](epic.md): major product epics organized by user value.
- [Backlog](backlog.md): prioritized implementation backlog.
- [Open Questions](open-questions.md): unresolved product and technical decisions.

## Current Direction

Brainspill is planned as a desktop-first, account-backed, collaborative infinite canvas for creative moodboards and project whiteboarding. The recommended stack is Next.js, Convex, and Yjs.
